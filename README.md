# PPTVbaAssist

## 機能
PPTのVBA開発を支援するツールです。  
VBAコードの効率的な入出力や管理を可能にします。

## インストール方法

インストーラーを使用するか、手動でアドインを登録してください。

### インストーラーを使用する場合（推奨）
1. リリースページから最新の `PPTVbaAssist_Setup_x.x.x.exe` をダウンロードして実行します。
2. インストール完了後、PowerPointを起動すると自動的にメニューにアドインが追加されます。

### 手動でインストールする場合
1. リリースページから **PPTVbaAssist.ppam** をダウンロードします。  
2. PPTの「アドイン」機能を使用して、ダウンロードした `.ppam` ファイルをアドインとしてインストールしてください。

> **Tips:**
> PowerPoint上で「ALT」→「T」→「I」キーを順番に押すと、アドインダイアログを素早く開くことができます。このダイアログでアドインの追加・削除や、一時的な有効化/無効化の切り替えが可能です。

## 主なコンポーネント

### PPTVbaAssist.ppam
- VBAソースコードの入出力を簡略化します。  
- `src`フォルダを介してソースコードとのやり取り（インポート/エクスポート）を行います。
